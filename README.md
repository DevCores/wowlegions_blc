## Server Requirements
<ul>
<li>PHP >= 7.1.3
<li>OpenSSL PHP Extension
<li>PDO PHP Extension
<li>Mbstring PHP Extension
<li>Tokenizer PHP Extension
<li>XML PHP Extension
